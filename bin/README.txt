The directory contains utility scripts and programs used in development

convertXML2HTML20221209.py reads from toc.xml and map.jhm in java code and creates a toc.html file; it also copies all the html files and corrrespning images.

copy_nbm copies all the nbm files in the src directory to the target directory.

updateCommandLineHelpInSeaDASGUI.py  and updateExtractorsHelpInSeaDASGUI.py replace the command line help in SeaDAS GUI with the current tag.

GenAUCatalog.class creates updates.xml and updates.xml.gz from a certain directory.
