About OCSSW Client Server Module

Introduction

The OCSSW Client-Server Module provides a platform for Windows OS users to remote access to the OCSSW package that is installed in a non-windows environment from the SeaDAS appcilation installed on Windows.

Directory Structure

This is the structure of the OCSSW Client-Server (seadas-ocsswrest) package:

  • bin - SeaDAS executable and SeaDAS command-line tools.
  • config - SeaDAS configuration file.
  • lib - A text file that keeps change logs.
  • modules - License file.
  • licenses - projec object model(pom) file.
  • modules - Readme file.
  • modules - Current version of the module.
