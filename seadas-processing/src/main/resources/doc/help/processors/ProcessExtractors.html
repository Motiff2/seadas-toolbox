Product Extractors

Creates an extracted data product.

Location

Tools -> OCSSW Processing -> extractor

Details

The Extractor GUI serves as a front-end to several product extractors provided with SeaDAS:

  • l1aextract_seawifs
  • l1aextract_modis
  • l2extract

You can access the extractor GUI from the Tools/OCSSW Processing/extractor menu. The extractor GUI initially displays a default window, as shown in the image below, to accept input parameters/options.

Once an infile is selected, the program will decide which extractor to be used based on the input file type and display customized GUI for each specific extractor.
