OCSSW Configuration

Configures OCSSW installation.

Location
Details

Location

Tools -> SeaDAS Processing -> OCSSW Configuration

Details

OCSSW Branch - this is the git branch of the OCSSW software. This is only 2 digits whereas the SeaDAS versioning scheme is 3 digits. In example SeaDAS 7.5.2 and 7.5.3 would both have OCSSW branch 7.5 as it's default settings. If a branch is selected which does not correspond with the SeaDAS version (such as choosing OCSSW branch 7.5 while using SeaDAS 7.4.x) then a possibility could exist of problems running OCSSW from the SeaDAS GUI.

OCSSW Location - this is the physical location of the OCSSW software which you intend to use when running from the SeaDAS GUI. The following additional parameters are needed dependent on the selection of the OCSSW Location parameter:

  • local - OCCSW is physically located on the machine (not applicable to Windows operating systems)
    • OCSSW ROOT - the directory where OCSSW is installed
  • virtualMachine - OCSSW is located on the virtual machine
    • OCSSW Shared Dir - the directory where OCSSW is installed
  • remoteServer - OCSSW is located on a remote machine
    • OCSSW Server Address
    • Server Port
    • Server Input Stream Port
    • Server Error Stream Port

Note: this tool updates the configuration file: ~/.snap/etc/seadas.properties
