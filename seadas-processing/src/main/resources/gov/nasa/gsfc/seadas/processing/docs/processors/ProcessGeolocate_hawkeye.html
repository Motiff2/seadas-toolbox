Details

geolocate_hawkeye creates a GEO file from HAWKEYE L1A input files.

For geolocate_hawkeye to run properly, it needs the DEM files. The DEM files for HAWKEYE are on NASA OBPG FTP server:

ftp://samoa.gsfc.nasa.gov/pub/HAWKEYEL1/DEM-LWM.tar.gz

Extract the tar file in the $OCSSWROOT/run/data/hawkeye directory. The tar file is 5G, and the extracted file size will be 52G.
Point the terrain_path to the location of the extracted tar file.
